『会社までの距離が遠く、面接時間は1時間以上で、質疑応答が少なめだと、もっとも受かりやすい』という意味です。
scoreが12というのは、AからOまでの面接で合格しているものについて 1 1 0 なら1点、不合格のものについて 1 1 0 でなければ1点として、合計した値のことです。

これは、過去の面接では、1 1 0 の条件が揃った面接は合格し、不合格になったことはない、ということを意味しています。
AIは、1 1 0 の条件が揃った面接であれば、落ちることはなく、受かるはずだから安心しなさい、と言っています。
